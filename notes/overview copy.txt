django setup:

    Install virtual enviorment:

        in RealtimeChat folder do this command to install virtualenv
            brew install virtualenv 

    Django virtual enviorment setup:

        in RealtimeChat folder do one of these command to create a virtual environment
            virtualenv -p python3 venv
            python3 -m venv venv

        in RealtimeChat folder do this command to activate the virtual enviorment
            source venv/bin/activate

            terminal will show (venv)
                (venv) chetrambassit@Chetrams-Mac-mini RealtimeChat % 

    Django installation:
    
        In RealtimeChat folder do this command to install django 
            pip install django 


    Create requirements.txt file to view packages installed 

        pip freeze > requirements.txt 

        update requirements.txt file 
            pip freeze 

    
    create a django project 

        django-admin startproject <project-name>
        we will call this project core

added react native encrypted storage 
    React Native Encrypted Storage is a library used to securely store sensitive data on a device in a React Native app.

    It lets you store things like:
        login credentials (tokens, passwords)
        API keys
        session data

    …but instead of saving them as plain text, it encrypts them using the device’s secure storage systems:
        iOS → Keychain
        Android → Encrypted SharedPreferences / Keystore

    👉 Encrypted storage protects against that:
        data is encrypted at rest
        tied to device security
        harder to extract


added react native image picker 
    React Native Image Picker is a library that lets your app access a user’s photos or camera so they can select or take images/videos.

    🧠 What it actually does
        It gives your app the ability to:
            Open the photo gallery
            Launch the camera
            Let the user pick or capture an image/video
            Return that file back to your app (with URI, file name, type, etc.)

    📸 Example flow
        User taps “Upload Profile Picture”
        Image picker opens (camera or gallery)
        User selects/takes a photo
        Your app gets the image data
        You display or upload it


    follow instructions here:
        https://github.com/react-native-image-picker/react-native-image-picker
